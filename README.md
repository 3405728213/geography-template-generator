# 🌍 高中地理大题答题模板生成器

一款基于 AI 的 Web 应用，帮助高中地理老师自动生成规范化的高考大题答题模板。

## ✨ 功能特性

- **5 大题型全覆盖**：成因分析、影响评价、特征描述、措施建议、比较类
- **AI 智能生成**：接入 Claude API，根据输入的题目自动生成答题模板
- **结构化输出**：每题包含答题思路、模板框架、关键术语、完整示例、得分要点
- **填空式模板**：生成可直接套用的模板，用【___】标注需填写内容
- **复制 & 打印**：一键复制全文或打印为讲义
- **历史记录**：自动保存最近生成记录，方便复用
- **响应式设计**：适配电脑、平板，适合课堂投影

## 🚀 快速开始

### 1. 环境要求

- Python 3.8+
- Anthropic API Key（[申请地址](https://console.anthropic.com/)）

### 2. 安装依赖

```bash
cd geography-template-generator
pip install -r requirements.txt
```

### 3. 配置 API Key

```bash
# 复制配置模板
copy .env.example .env

# 编辑 .env 文件，填入你的 API Key
# ANTHROPIC_API_KEY=你的API密钥
```

### 4. 启动应用

```bash
python app.py
```

浏览器打开 `http://localhost:5000` 即可使用。

## 📖 使用说明

1. 左侧**选择题型**（成因分析/影响评价/特征描述/措施建议/比较类）
2. 选择**子类型侧重**（如自然原因/人为原因）
3. 在输入框中**输入题目或知识点**（例如："分析黄土高原水土流失的原因"）
4. 选择知识模块和模板难度
5. 点击「**生成答题模板**」
6. 右侧即显示 AI 生成的规范化答题模板
7. 可**复制**全文或**打印**为讲义

## 🎯 题型覆盖

| 题型 | 子类型 | 模板结构 |
|------|--------|---------|
| 🔍 成因分析 | 自然原因 / 人为原因 / 综合分析 | 气候→地形→水文→土壤→生物→人类活动 |
| ⚖️ 影响评价 | 有利/不利 / 经济/社会/生态 | 辩证分析 + 综合评价 |
| 📊 特征描述 | 分布/变化/结构特征 | 总体→具体→极值→归纳 |
| 🛠️ 措施建议 | 工程/生物/技术/管理措施 | 问题诊断→分类措施→可行性 |
| 🔄 比较类 | 异同/区位/区域比较 | 维度拆解→相同点→不同点→综合 |

## ⚙️ 模型选择

| 模型 | 速度 | 质量 | 成本 | 适用场景 |
|------|------|------|------|---------|
| `claude-haiku-4-5` | ⚡快 | ⭐⭐⭐ | 💰低 | 日常备课、快速生成 |
| `claude-sonnet-4-6` | 适中 | ⭐⭐⭐⭐⭐ | 💰💰中 | 重要考试、进阶模板 |

在 `.env` 文件中修改 `CLAUDE_MODEL` 即可切换。

## ⚠️ 注意事项

- 生成的模板基于 AI 模型，建议使用前人工审核知识点准确性
- 模板仅供教学参考，实际考试请以官方评分标准为准
- 需要稳定的网络连接以访问 Claude API
